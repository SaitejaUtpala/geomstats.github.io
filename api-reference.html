API reference¶

The “manifold” module¶

Manifold, i.e. a topological space that locally resembles Euclidean space near each point.

class geomstats.manifold.Manifold(dimension)[source]¶

Class for manifolds.

belongs(point, point_type=None)[source]¶

Evaluate if a point belongs to the manifold.

regularize(point, point_type=None)[source]¶

Regularize a point to the canonical representation chosen for the manifold.

The “hypersphere” module¶

The n-dimensional hypersphere embedded in the (n+1)-dimensional Euclidean space.

class geomstats.hypersphere.Hypersphere(dimension)[source]¶

Class for the n-dimensional hypersphere embedded in the (n+1)-dimensional Euclidean space.

By default, points are parameterized by their extrinsic (n+1)-coordinates.

belongs(point, tolerance=1e-06)[source]¶

Evaluate if a point belongs to the Hypersphere, i.e. evaluate if its squared norm in the Euclidean space is 1.

extrinsic_to_intrinsic_coords(point_extrinsic)[source]¶

Convert from the extrinsic coordinates in Euclidean space, to some intrinsic coordinates in Hypersphere.

intrinsic_to_extrinsic_coords(point_intrinsic)[source]¶

Convert from the intrinsic coordinates in the Hypersphere, to the extrinsic coordinates in Euclidean space.

projection(point)[source]¶

Project a point on the Hypersphere.

projection_to_tangent_space(vector, base_point)[source]¶

Project a vector in Euclidean space on the tangent space of the Hypersphere at a base point.

random_uniform(n_samples=1, bound=0.5)[source]¶

Sample in the Hypersphere with the uniform distribution.

random_von_mises_fisher(kappa=10, n_samples=1)[source]¶

Sample in the 2-sphere with the von Mises distribution centered in the north pole.

regularize(point)[source]¶

Regularize a point to the canonical representation chosen for the Hypersphere, to avoid numerical issues.
